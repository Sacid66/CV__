Ada Web Browser

Ada is a web browser built using Google's infrastructure and developed in Python by Sacid Heysem Sicak. This project was created to practice web browser development.

With Ada, you can perform quick searches on the internet and navigate easily with its simple and user-friendly interface. The browser's core features include fast searches and tabbed browsing.

The goal of this project is to learn the basics of building a web browser using Python and to gain experience in the process. Ada was developed with this objective in mind.
